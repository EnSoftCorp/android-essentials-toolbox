Installation

Installing the Android Essentials Toolbox is easy. The toolbox is installed as an Eclipse plugin. It is recommended to install the toolbox from the provided update site, but it is also possible to install from source.

Installing from update site

First make sure you have Atlas Standard or Pro version 2.0.2 or higher installed. You should also install the Android ADT Eclipse plugin. Then follow the steps below to install the Android Essentials Toolbox from the Eclipse update site.

  1. Start Eclipse, then select Help > Install New Software.

  2. Click Add, in the top-right corner.

  3. In the Add Repository dialog that appears, enter Android Essentials Toolbox for the Name and https://android-essentials-toolbox.github.io/updates/ for the Location.

  4. In the Available Software dialog, select the checkbox next to "Android Essentials Toolbox" and click Next followed by OK.

  5. In the next window, you'll see a list of the tools to be downloaded. Click Next.

  6. Read and accept the license agreements, then click Finish. If you get a security warning saying that the authenticity or validity of the software can't be established, click OK.

  7. When the installation completes, restart Eclipse.

Installing from source

If you want to install from source for bleeding edge changes, follow the instructions at the source repository.
